## 💖 What This Is

A mobile-first affirmation experience triggered by an NFC tag.

One tap.  
One ritual.  
One randomized combo of:

- 🎵 Spotify hype track  
- 🖼️ Aesthetic artwork  
- 💬 Flashing main-character affirmation  

Built as a soft but powerful daily activation moment.

---

## ✨ Tap the NFC → Ritual Activated

1. Tap your NFC tag  
2. Browser opens instantly  
3. Sparkle animation loads  
4. A random:
   - Song  
   - Image  
   - Affirmation  
   appears  
5. Music begins playing  
6. New Hype button lets you refresh without repetition  

It is intentionally designed to:
- Avoid consecutive repetition
- Feel different every time
- Load smoothly on mobile
- Feel slightly magical ✨
